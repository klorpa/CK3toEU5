This program will convert a Crusader Kings 3 save game to a Europa Universalis 5 mod.

----
IMPORTANT LINKS
----

Forum thread: https://forum.paradoxplaza.com/forum/threads/ck3-to-eu5-converter-thread.1866691/
GitHub project: https://github.com/ParadoxGameConverters/CK3toEU5
Documentation: N/A

----
SUPPORTED VERSIONS
----
CK3 - 
EU5 - 


----
INSTRUCTIONS
----
1. Run ConverterFrontend.exe
2. Set all the options, making sure the directories are correct.
3. Click Convert
4. Start Europa Universalis 5 and select the playset with your new mod.
